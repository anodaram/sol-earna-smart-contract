# sol-earna-smart-contract

## Smart Contract
Built on Anchor Framework

### Build & Test
``` bash
anchor build

anchor test
```


### Deploy
Set network
```
# set to localhost
solana config set --url localhost

# set to devnet
solana config set --url devnet
```

Deploy Script
``` bash
anchor deploy
```
